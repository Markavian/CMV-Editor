CMV Movie Editor

About

This application lets you edit CMV Movie files created from the game Dwarf Fortress.

Dwarf Fortress is available from http://www.bay12games.com/dwarves/

The CMV file format is a compressed ASCII playback that uses a tileset or character set for playback. It is primarily used to export recordings of games in a lossless format to be played back.

An online flash player and supporting archive site for CMV files can be found at the Dwarf Fortress Map Archive - http://mkv25.net/dfma/; over a thousand player created movies can be found there.

Here follows a summary of the features explaining the available tools.

Main Interface

The CMV Movie Editor (CMVEditor) can load, save and create CMV and CCMV files. The main interface is divided into several areas; Stage Area, Movie Info, Playback Controls, Timeline, Paint Tools, Timeline Tools.

Stage Area

The stage area displays the active frame marked by the playhead on the timeline. When you load a movie file, or create a blank file, the first frame is displayed.

The stage area will update as you play through a movie, using the Playback Controls, and will respond to mouse clicks from Paint Tools.

Movie Info

The movie info box will display information about the currently loaded movie. If you have created a new movie from scratch (using File > New...) then the movie file name will be reported as "unsaved". 
File: sillyscout.cmv
Version: 10001
Size: 80 by 25
Frames: 2
Sounds: 0
  • File: The file currently being edited. "Unsaved" if unsaved.
  • Version: The fileformat of the loaded CMV file. Usually 10001.
  • Size: The dimensions of the first frame of the CMV file in Columns and Rows.
  • Frames: The total number of frames, will vary as you modify the timeline.
  • Sounds: The number of sound channels; not much is known about sounds, but if they're there, their number will be displayed.
